Just a normal webpage.

A unique webpage that lets you scroll to the left side!

How to do it:

Type this in your CSS code:

html, body {
   overflow: hidden;
}
.container {
   transform: rotateY(180deg);
   height: 100vh;
   width: 100%;
   overflow-y: scroll;
   overflow-x: hidden;
   position: fixed;
   top: 0;
   left: 0;
}
.container .content {
   transform: rotateY(180deg);
}

Why is it so cool?

Because this is a sooo... Unique Styled Website!
Browsers never made the scrollbar go to the left before... And now it does only on this website!
